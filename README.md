# Movies API Client

This is a front end for [https://github.com/0789oleh/movie-api]{Movies API}.


## Getting Started 

1. Clone repo

```bash
git clone <твой-репозиторий>
cd movies-spa
npm install

```

2. Launch the backend. Make sure that backend is working

```bash
http://localhost:8000/api/v1
```

3. Launh the frontend

```bash
npm run dev
```
